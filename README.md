# Recipe Inventory AI

A web prototype for generating recipes from your available groceries and user preferences.

## Setup

1. Install dependencies:
   - `npm install`
2. Run development server:
   - `npm run dev`
3. Open the local URL shown in the terminal.

## MVP features

- Manual ingredient entry
- Recipe generation based on inventory and preferences
- Preferences for effort, dietary tags, and banned ingredients
- Placeholder for receipt and fridge scan integration

## Future work

- Receipt OCR and fridge image scanning
- Inventory freshness and age prioritization
- AI-powered recipe generation
- More detailed dietary and appliance constraints
